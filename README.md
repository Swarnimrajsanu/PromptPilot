# PromptPilot 

An AI agent application built with Node.js.

## Features
- AI Agent implementation using OpenRouter/OpenAI API.
- Browser automation & testing with Playwright.

## Installation & Setup

1. Clone the repository:
   ```bash
   git clone https://github.com/Swarnimrajsanu/PromptPilot.git
   cd PromptPilot
   ```

2. Install dependencies (uses pnpm):
   ```bash
   npm install -g pnpm
   pnpm install
   ```
   *(Alternatively, you can use `npm install`)*

3. Set up environment variables:
   Create a `.env` file in the root directory and add necessary keys (e.g., `OPENROUTER_API_KEY`).

4. Run the project:
   ```bash
   node index.js
   ```

## Technologies Used
- Node.js
- Express
- Playwright
- OpenAI / OpenRouter SDK
- Zod
